# 🌐 Web Portfolio

Website portfolio pribadi yang dibuat untuk menampilkan project, skill, dan informasi kontak.

## 🚀 Tech Stack

- Next.js
- React
- Tailwind CSS
- TypeScript

## ✨ Fitur

- Tampilan responsive (mobile & desktop)
- Animasi sederhana
- Halaman portfolio project
- Form kontak

## 📦 Cara Menjalankan

```bash
npm install
npm run dev
```

## 🔐 Environment Variables

Buat file `.env.local` dan isi:

```env
GEMINI_API_KEY=your_api_key
RESEND_API_KEY=your_api_key
```
